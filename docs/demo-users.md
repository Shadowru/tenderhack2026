### 1. ГБУ Здравоохранения Москвы — Крупная больница
**ИНН:** `7734091519`

| Параметр | Значение |
|----------|---------|
| Контракты | 3 408 |
| Уникальных СТЕ | 2 210 |
| Категорий | 1 002 |
| Топ-3 направления | Лекарственные препараты, наркотические средства, медицинское оборудование |

**Почему выбран для демо:**
Крупнейший по охвату категорий покупатель в наборе — 1 002 уникальные категории. Профиль закупок максимально разнообразен: рядом с кетамином и наркотическими анальгетиками соседствуют канцелярские товары и хозяйственный инвентарь. Это идеальный стресс-тест для системы: покупатель, у которого «есть всё», должен при этом получать медицинский контекст с высоким приоритетом.

**Ожидаемое поведение поиска:**
- Запрос **«бумага»** → в топе медицинская бумага (рулонная, для ЭКГ, термобумага), офисная бумага ниже.
- Запрос **«перчатки»** → смотровые/хирургические перчатки выше хозяйственных.
- Запрос **«насос»** → инфузионные и аспирационные насосы выше промышленных.
- Запрос **«игла»** → медицинские иглы выше швейных.

### 2. ГБ ПОУ — Московский колледж
**ИНН:** `7716237684`

| Параметр | Значение |
|----------|---------|
| Контракты | 3 688 |
| Уникальных СТЕ | 3 233 |
| Категорий | 475 |
| Топ-3 направления | Канцелярия, хозяйственные товары, продукты питания |

**Почему выбран для демо:**
Абсолютный рекордсмен по числу уникальных СТЕ (3 233) при относительно небольшом количестве категорий (475). Профиль типичного крупного образовательного учреждения с широкой «длинным хвостом» номенклатуры: закупается буквально всё — от писчей бумаги до пищевых добавок. Хорошо демонстрирует, как персонализация работает в условиях высокой горизонтальной диверсификации.

**Ожидаемое поведение поиска:**
- Запрос **«бумага»** → офисная бумага для принтеров и тетради выше медицинской.
- Запрос **«молоко»** → пастеризованное молоко для питания студентов, не реагенты.
- Запрос **«стол»** → учебные/офисные столы, не операционные.
- Запрос **«программа»** → учебные программы/ПО для обучения, не медицинские системы.

### 3. ГБУ Культуры — Московская библиотека
**ИНН:** `7717038138`

| Параметр | Значение |
|----------|---------|
| Контракты | 1 598 |
| Уникальных СТЕ | 1 415 |
| Категорий | 498 |
| Топ-3 направления | Канцелярские товары, расходные товары |

**Почему выбран для демо:**
Культурное учреждение с узкой профессиональной специализацией. Несмотря на почти 500 категорий, основная масса закупок сосредоточена на канцелярии и расходниках — бумага, картриджи, папки, скоросшиватели. Позволяет проверить, насколько система умеет «видеть» библиотечную специфику: закупки книг, обложек, систем хранения документов.

**Ожидаемое поведение поиска:**
- Запрос **«бумага»** → бумага для печати и ксерокопирования, бумага для обёртки книг.
- Запрос **«обложка»** → обложки для книг и документов выше медицинских карт.
- Запрос **«картридж»** → стандартные офисные картриджи, не специализированные.
- Запрос **«стеллаж»** → книжные/архивные стеллажи выше складских.

### 9. ГБУ ЗО Пермского края — Больница
**ИНН:** `5902290473`

| Параметр | Значение |
|----------|---------|
| Контракты | 3 076 |
| Уникальных СТЕ | 1 952 |
| Категорий | 523 |
| Топ-3 направления | Лекарственные препараты, наркотические средства |

**Почему выбран для демо:**
Единственный не-московский покупатель в наборе (Пермский край). Медицинский профиль аналогичен пользователю №1, что позволяет сравнить поведение персонализации для двух похожих учреждений из разных регионов. Также позволяет проверить отсутствие географической предвзятости в системе: пермская больница должна получать те же медицинские приоритеты, что и московская.

**Ожидаемое поведение поиска:**
- Запрос **«бумага»** → медицинская бумага (рулонная, для ЭКГ) аналогично московской больнице.
- Запрос **«препарат»** → лекарственные препараты, не бытовые товары.
- Запрос **«шприц»** → медицинские одноразовые шприцы однозначно в топе.
- Сравнение с пользователем №1: результаты поиска должны быть схожи, несмотря на разные ИНН и регион.

### 10. ГБУ ЗО — Диагностическая лаборатория
**ИНН:** `7704054943`

| Параметр | Значение |
|----------|---------|
| Контракты | 1 439 |
| Уникальных СТЕ | 1 260 |
| Категорий | 465 |
| Топ-3 направления | Стандартные образцы, канцелярия |

**Почему выбран для демо:**
Наиболее специализированный профиль в наборе. Основная номенклатура — реагенты, стандартные образцы, калибровочные материалы — практически уникальна среди всех покупателей. Хороший тест на «длинный хвост»: система должна корректно работать с редкими и нишевыми товарными категориями, не смешивая лабораторные реагенты с бытовой химией.

**Ожидаемое поведение поиска:**
- Запрос **«образец»** → стандартные образцы и эталонные материалы для лабораторий выше образцов тканей или шаблонов документов.
- Запрос **«раствор»** → лабораторные буферные/калибровочные растворы выше бытовых чистящих средств.
- Запрос **«пробирка»** → лабораторные пробирки однозначно в топе.
- Запрос **«бумага»** → индикаторная/фильтровальная бумага и офисная бумага примерно равнозначны — интересный граничный случай для отладки.
